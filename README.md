# ConatusGames

Plataforma web de jogos interativos desenvolvida através da metodologia de Vibe Coding com Claude Opus 4.1, demonstrando o potencial da programação assistida por inteligência artificial.

## Sobre

Este repositório documenta a criação de uma coleção de jogos web utilizando a abordagem moderna de Vibe Coding, onde a inteligência artificial atua como linguagem de programação de alto nível. O projeto exemplifica como prompts em linguagem natural podem gerar aplicações funcionais e interativas, acelerando o processo de desenvolvimento sem comprometer a qualidade.

## Estrutura do Repositório

O projeto está organizado nos seguintes componentes principais:

**Interface Principal**  
Página inicial responsiva com navegação intuitiva entre os jogos disponíveis, desenvolvida com HTML5, CSS3 e JavaScript moderno. A interface adapta-se automaticamente a diferentes dispositivos e tamanhos de tela.

**Coleção de Jogos**  
Módulos independentes contendo diferentes jogos interativos, cada um com sua própria lógica, controles e mecânicas. Os jogos estão organizados no diretório `jogos/` para facilitar manutenção e expansão.

**Recursos Visuais**  
Assets gráficos incluindo animações, sprites e elementos visuais que enriquecem a experiência do usuário, armazenados no diretório `gifs/`.

## Organização dos Arquivos

O repositório contém:

- **index.html**: Estrutura da página principal e ponto de entrada da aplicação
- **style.css**: Folha de estilos global com design responsivo e temas visuais
- **script.js**: Lógica de navegação, controles e funcionalidades compartilhadas
- **jogos/**: Diretório contendo os módulos individuais de cada jogo
- **gifs/**: Recursos visuais e animações

## Metodologia de Desenvolvimento

O projeto foi desenvolvido utilizando a metodologia Vibe Coding, caracterizada por:

**Programação por Intenção**  
Descrição de funcionalidades através de linguagem natural, permitindo que a IA compreenda e implemente a lógica necessária sem necessidade de código explícito inicial.

**Iteração Ágil**  
Ciclos rápidos de desenvolvimento, teste e refinamento, onde cada iteração adiciona novas funcionalidades ou melhora as existentes baseado em feedback imediato.

**Colaboração Humano-IA**  
Sinergia entre criatividade humana e capacidade de geração de código da IA, onde o desenvolvedor mantém controle criativo enquanto a IA acelera a implementação técnica.

## Tecnologias Utilizadas

**Frontend**
- HTML5: Estruturação semântica das páginas
- CSS3: Estilização moderna com flexbox, grid e animações

## Como Utilizar

**Acesso Local**
1. Clone o repositório: `git clone https://github.com/Agrippa-Tech/ConatusGames.git`
2. Navegue até o diretório: `cd ConatusGames`
3. Abra `index.html` diretamente no navegador ou utilize um servidor local

**Servidor Local (Opcional)**
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000
```

Acesse através de `http://localhost:8000` no navegador de sua preferência.

## Vibe Coding: Conceito e Aplicação

Vibe Coding representa uma mudança de paradigma no desenvolvimento de software, popularizado por Andrej Karpathy. Nesta abordagem, o programador utiliza linguagem natural como interface de programação de alto nível, delegando à IA a tarefa de traduzir intenções em código funcional.

**Princípios Fundamentais**
- Foco na lógica e funcionalidade desejada, não na sintaxe
- Iteração rápida através de refinamentos incrementais
- Aproveitamento das capacidades de compreensão contextual da IA
- Manutenção de controle criativo pelo desenvolvedor humano
